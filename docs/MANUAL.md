# マニュアル　永い後日談のネクロニカ
v0.2.0

## PC（ドール）

### 初期作成
- キャラクター作成時、基本パーツ、「たからもの」のみを保有したドールが作成されます。

### パーツ操作
- 部位単位でのパーツ一括回復／破壊（Ctrl+クリック）
- 部位単位でのスキル使用解除
- パーツによる判定
- パーツによる攻撃判定
- パーツの使用宣言

### キャラクターインポート
- [キャラクター保管所](https://charasheet.vampire-blood.net/)のキャラクターシートから変換用のマクロが付属しています。

## 戦闘
- 戦闘開始時、現在行動値が最大行動値に変更されます。
- ラウンド経過時、最大行動値の値分、現在行動値が回復します。
- 戦闘参加キャラクターの現在行動値が変動した際、イニシアティブが変更されます。

## ツールチップ表示
- トークンにカーソルを合わせた際、ツールチップによりパーツ一覧が表示されます。
  - 部位「任意」はツールチップに表示されません。（いずれかの部位に変更することを想定しています。）
